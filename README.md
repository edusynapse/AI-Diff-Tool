# AI-Diff-Tool

A desktop app built with Electron and Node.js to apply diff patches to JS data storage files using the xAI Grok 4 API.

## Setup
1. Clone: `git clone https://github.com/yourusername/AI-Diff-Tool.git`
2. `cd AI-Diff-Tool`
3. `npm install`
4. `npm start`

## Usage
- Paste/load diff patch into top textarea.
- Paste/load JS model content into bottom textarea (or use samples).
- Enter xAI API key (get from https://console.x.ai).
- Click "Apply" â€“ preview modified JS in output area and download.
